多机切换 · 页面初版原型

每个页面一个独立 HTML（评审样机，无 emoji，左侧栏对齐现有实现）。点卡片进入； 页内左侧导航为真实跳转，机器切换器选中的当前机器跨页保持（localStorage 模拟全局 currentNode）。 寻址 /n/<nodeId>/...、Broker-local /api/broker/*。配套文档见 ../多机切换交互设计.md。
