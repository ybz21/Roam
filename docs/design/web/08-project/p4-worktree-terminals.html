规则：① 命令行 = 会话粒度（pane 归 tmux 自己管，只展示 pane 数）；挂靠关系 = cwd join 实况，会话 cd 走了自动离组、cd 进来自动入列；多 pane 命中多 worktree 显 ⚠（hover 列全部 matches，同 07 §2.4 ambiguous）；② 新开命令行：shell = ttmux new --dir <worktree>（纯目录语义，命名 <分支>-sh 自动后缀）；Claude/Codex = POST /worktree-sessions「已有」档（注入 cd + agent 启动）——孤儿由此复活、外部由此收编；③ 输出预览 = tmux capture-pane 尾行，展开的 worktree 才拉取（懒加载 + 5s 缓存），只读不交互；④ 「进入」→ 右侧停靠终端（手机全屏），工具栏面包屑 项目 › ⎇ 分支 › 命令行，点分支段回本 tab；⑤ 删除按钮在有命令行挂靠时置灰（占用检查前置到 UI，后端 Remove 仍兜底）；⑥ 「派生」= fork 表单（parent 固定，默认同 worktree）。
