# Roam — 功能列表

[English](features.md) ｜ **简体中文**

Roam 的详细能力清单。总览与截图见 [README](../README.zh-CN.md)。

## 远程接入 & 多设备
- **零安装 Web 控制台** —— 手机、平板、笔电任意浏览器打开即用，客户端不用装任何东西。
- **换设备回到同一现场** —— 断网、关浏览器、换设备后重连，落回你离开时的位置。
- **服务端偏好 + 跨设备同步** —— 主题、语言、布局、快捷命令跟着你走。
- **终端状态随 URL 恢复** —— 打开的终端/标签编码进 hash，刷新或分享链接能重开同一组。
- **PWA 装到主屏** —— 可加到主屏（192px 图标），像 App 一样启动。
- **对远程接入友好** —— 配合 Tailscale / Cloudflare Tunnel / frp / SSH 转发；默认自签 HTTPS，手机经 LAN/隧道用麦克风、剪贴板都在安全上下文里。
- **自动升级检测** —— 检测到远端有新构建时弹更新提示条。

## 会话 & 终端
- **持久终端（基于 tmux）** —— 终端、开发服务、长任务留在开发机上，断开也继续跑。
- **多标签终端** —— 字号调节、复制、扩展按键条、断线自动重连。
- **会话列表 / 筛选 / 搜索** —— 按 Claude / Codex / 蜂群 / 状态（待确认、空闲）筛选；两行筛选条带搜索；按最近活跃排序。
- **会话重命名**；每个会话带 `last_activity` 最近活跃时间。
- **独立终端页** —— 单个会话全屏开在独立标签页（`#/term/<名>`）。
- **新建会话支持 Git Worktree 模式** —— 在独立 worktree/分支上起会话。
- **快捷命令** —— 存常用命令，一键发送，点完焦点回输入框（不重复触发），刷新后保留。
- **TUI Agent 里可拖选复制** —— Claude/Codex 备用屏会话里鼠标能选中文本（过滤掉应用鼠标上报，不影响选择）。
- **全屏 TUI 翻历史** —— 鼠标滚轮在 Claude/Codex 备用屏里上翻（后端合成滚轮事件）。
- **手机按键条 + 独立输入框** —— 屏幕按键（Enter、Esc、Tab、方向键…）加独立输入框，软键盘不会吃掉你的「发送」。

## AI 编码 Agent
- **Claude Code、Codex 一等公民** —— 启动、命名、分组、追踪、收集输出；会话列表里带 Agent 标签。
- **Agent 状态感知** —— 识别「等确认」「空闲」「运行中」。
- **对话视图** —— Agent 之上的会话式 UI，带块渲染、复制、提示状态检测。
- **粘贴图片到终端 / 对话** —— 截图直接粘进输入框（自动加 `@` 前缀作文件引用）；Shift+Enter 换行。
- **随处追加指令** —— 从任意设备补充要求，不丢上下文。
- **可配置启动命令** —— 自定义拉起 Claude / Codex 的 shell 命令。

## 蜂群编排（swarm）
- **把一个目标分给多个成员** —— API 给一个、前端给一个、测试给第三个。
- **成员引擎与角色** —— 每个成员可选 Claude 或 Codex；master/worker 角色，细分角色（产品/工程/测试…）+ 职责。
- **自动 Leader** —— 用模板拉起 Leader，避免单干；每个蜂群支持工作目录。
- **提示词模板** —— 基于 Go `text/template` 的成员提示词，按 markdown 渲染。
- **实时依赖拓扑** —— 动画化的实时拓扑（「办公室」视图），移动端自适应。
- **协作墙（广场）** —— 人与各 Agent 的共享消息流；监听循环让 master/worker 持续盯 human 指令与广场消息。
- **可拖拽看板** —— 待办 / 已派 / 进行 三列看板；待处理 Inbox 收需人工介入的事项。
- **依赖与解锁** —— 完成的活儿解锁下一步；完成需显式标记。
- **隔离** —— 蜂群会话对常规会话列表和 Go CLI 原生列表隐藏。

## 文件 & 编辑
- **文件浏览器** —— 平铺视图 + VS Code 式树，一键切换；隐藏文件开关。
- **排序与导航** —— 按 名称 / 种类 / 修改日期 / 创建日期 / 大小 排序；路径栏上方有最近目录快捷按钮。
- **全局右键菜单** —— 统一的文件右键菜单（打开、删除、引用进终端…）。
- **Monaco 代码编辑器** —— `FileWorkspace` 组件，VS Code 式双栏编辑；把文件或标签拖到右半区分栏（编辑器区 + 终端区）。
- **实时重载** —— 文件被外部工具改动后，打开的编辑器里自动重载。
- **内联预览** —— Markdown（含 **Mermaid** 图渲染）、图片、office/文件预览、相对链接；按类型显示文件图标。
- **拖文件进终端** —— 把该文件的引用丢进输入框。

## 浏览器操控
- **实时浏览器镜像** —— 开发机上的 Chrome 串流进控制台：开标签、导航、点击、输入。
- **标签栏** —— 列出/切换/关闭被镜像 Chrome 的标签；可指定镜像哪个标签。
- **画质控制** —— 自动 / 标清 / 高清 / 超清，带延迟、带宽、帧率读数；打字/加载时画面不抖。
- **Chrome 配置面板** —— 有头/无头切换、窗口尺寸、profile、二进制路径、缩放，全在设置里。
- **稳健接入** —— 附着已在调试端口的 Chrome，或自己拉起；CDP 端口被占自动换；启动失败原因回传 UI。

## 手机操控
- **Android / iOS 真机镜像** —— 连着的真机串流进控制台，带远程导航栏和应用启动器。
- **自适应推流** —— 画质分档带延迟/带宽/帧率；三端（宿主/Android/iOS）连接配置。
- **按需装依赖** —— Android 工具链用到才装（平台开关）。
- **复现移动端流程** —— 在终端旁边顺手看个 App 或移动网页。

## 语音
- **语音输入** —— 对着 Agent/终端说话。
- **ASR 选路** —— 豆包/火山（标准版与极速版按 `resourceId` 自动选）或 OpenAI 兼容端点；麦克风需安全上下文，故默认 HTTPS。

## 插件 & 可扩展
- **内置插件** —— **IM 常驻管家 concierge**（飞书自建应用双向机器人：`@机器人` 派活、Agent 主导对话、复杂活委派 worker；经 provider 适配可扩展钉钉）、**reviewmesh**（空闲时 Codex+Claude 自动互审，产出 finding 与质量闭环）。
- **插件机制 v1** —— 带 manifest 的插件包（node/exec 运行时）走安装流程，另有编译进二进制的 builtin Go 模块插件；公开 SDK、`plugin dev` 工具链、激活事件、watchers、每插件存储。见 [plugins/](../plugins/) 与 [docs/design/plugin/](design/plugin/)。

## 命令行 & 自动化
- **`ttmux`** —— 管理持久会话、后台任务、Agent worker、swarm 和机器可读状态。
- **`chrome`** —— 驱动开发机 Chrome，用于 UI 调试、截图、表单流程、自动验收（Playwright over CDP）。
- **可脚本化** —— 把会话、任务、日志、编排暴露给脚本和 Agent。

## 安全 & 访问
- **访问口令** + 可选 **TOTP 两步验证**；多次失败后锁定。
- **开箱自签 TLS** —— 根 CA + 叶子证书拆分，带「下载证书」入口方便设备信任。
- **按 SSH 级权限对待** —— 设计上跑在隧道/VPN 后，别直接暴露公网。

## 体验 & 国际化
- **中英双语 UI** —— 全产品强制 i18n 标准。
- **深色 / 浅色主题**；全站统一 `#/xxx` hash 路由；侧栏图标 + tooltip。
- **概览页** —— 会话优先排序、自适应布局、快捷统计（会话 / 蜂群 / 活跃成员 / 待解锁）。
- **主动请求剪贴板权限** —— 加载时就请求，避免用时才弹授权卡住。
- **人和 Agent 一起用** —— 人从控制台亲手接管，Agent 读状态、收输出、继续推进。
- **许可证** —— AGPLv3。
