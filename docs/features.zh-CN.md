# Roam — 功能列表

[English](features.md) ｜ **简体中文**

Roam 的能力清单。总览与截图见 [README](../README.zh-CN.md)。

## 远程接入 & 多设备
- **零安装 Web 控制台** —— 手机、平板、笔电任意浏览器打开即用，客户端不用装任何东西。
- **换设备回到同一现场** —— 断网、关浏览器、换设备后重连，落回你离开时的位置。
- **PWA 安装** —— 可加到主屏，像 App 一样启动。
- **对远程接入友好** —— 配合 Tailscale / Cloudflare Tunnel / frp / SSH 转发；默认自签 HTTPS，手机经 LAN/隧道用麦克风、剪贴板都在安全上下文里。

## 会话 & 终端
- **持久终端（基于 tmux）** —— 终端、开发服务、长任务留在开发机上，断开也继续跑。
- **多标签终端** —— 字号调节、复制、扩展按键条、断线自动重连。
- **会话列表 & 筛选** —— 按 Claude / Codex / 蜂群 / 状态（待确认、空闲）筛选；可重命名、分组、点进任意会话。
- **独立终端页** —— 单个会话全屏开在独立标签页（`#/term/<名>`）。
- **手机按键条** —— 屏幕上的按键（Enter、Esc、Tab、方向键…）加发送框，手机上也能给 Agent 敲指令。

## AI 编码 Agent
- **Claude Code、Codex 一等公民** —— 启动、命名、分组、追踪、收集输出。
- **Agent 状态感知** —— 识别 Agent 是「等确认」「空闲」还是「运行中」。
- **随处追加指令** —— 从任意设备补充要求，不丢上下文。
- **可配置启动命令** —— 自定义拉起 Claude / Codex 的 shell 命令。

## 蜂群编排（swarm）
- **把一个目标分给多个成员** —— API 给一个、前端给一个、测试给第三个。
- **实时依赖拓扑** —— 成员与依赖关系的实时图。
- **协作墙（广场）** —— 人与各 Agent 的共享消息流。
- **可拖拽看板** —— 待办 / 已派 / 进行 三列看板管理群内任务。
- **依赖与解锁** —— 完成的活儿解锁下一步；完成需显式标记。

## 文件 & 编辑
- **文件浏览器** —— 平铺视图 + VS Code 式树，一键切换。
- **控制台内预览与编辑** —— Monaco 编辑、Markdown/Mermaid 渲染、office/文件预览。
- **文件与终端并排** —— 在运行的 Agent 旁边浏览、打开文件。

## 浏览器操控
- **实时浏览器镜像** —— 开发机上的 Chrome 串流进控制台：开标签、导航、点击、输入。
- **典型场景** —— 调试网页、保住登录态、让 Agent 复现一整套流程，全在开发机上。
- **画质控制** —— 自动 / 标清 / 高清 / 超清，带延迟、带宽、帧率读数。

## 手机操控
- **Android 真机镜像（adb）** —— 连着的真机串流进控制台，带远程导航栏和应用启动器。
- **复现移动端流程** —— 在终端旁边顺手看个 App 或移动网页。

## 语音
- **语音输入** —— 对着 Agent/终端说话；ASR 走豆包/火山或 OpenAI 兼容端点（麦克风需安全上下文，故默认 HTTPS）。

## 插件 & 可扩展
- **内置插件** —— IM 常驻管家 concierge（飞书，可扩展钉钉…）、reviewmesh（空闲自动互审）。
- **插件机制** —— 带 manifest 的插件包（node/exec 运行时）走安装流程，另有编译进二进制的 builtin Go 模块插件；公开 SDK + `plugin dev` 工具链。见 [plugins/](../plugins/) 与 [docs/design/plugin/](design/plugin/)。

## 命令行 & 自动化
- **`ttmux`** —— 管理持久会话、后台任务、Agent worker、swarm 和机器可读状态。
- **`chrome`** —— 驱动开发机 Chrome，用于 UI 调试、截图、表单流程、自动验收。
- **可脚本化** —— 把会话、任务、日志、编排暴露给脚本和 Agent。

## 安全 & 访问
- **访问口令** + 可选 **TOTP 两步验证**。
- **开箱自签 TLS**；多次失败后锁定。
- **按 SSH 级权限对待** —— 设计上跑在隧道/VPN 后，别直接暴露公网。

## 体验 & 国际化
- **中英双语 UI** —— 全产品强制 i18n 标准。
- **深色 / 浅色主题**，偏好存服务端。
- **人和 Agent 一起用** —— 人从控制台亲手接管，Agent 读状态、收输出、继续推进。
